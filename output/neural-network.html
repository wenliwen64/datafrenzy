Neural Network¶

  1. Hypothesis Representation / Non-linear hypothesis

  2. why complex non-linear hypothesis:

    1. Some label distribution require non-linear;
    2. If the feature number is large, linear regression including non-linear terms would be computationally expensive, that is why we need neural network;

  3. Neuron model:

  4. input, bias unit, input wires, activation function(h_theta), output

  5. Neural network:

  6. input layer:
    $$x_0(bias unit), x_1, x_2, x_3...$$
  7. hidden layer(in between input and output layer):
    $$a_0^{(2)}, a_1^{(2)}, a_2^{(2)}, a_3^{(2)}...$$
  8. output layer: final layer;
  9. Mathematical definition how to represent hypothesis in neural network(matrix computation):
    $$a^{(j+1)} = g(\Theta^{(j)}a^{(j)})=g(z^{(j+1)})$$
    , input layer is
    $$a^{(1)}$$
    , g is sigmoid function.
  10. next layer takes this layers output as features.
  11. Use neural network to represent logic unit:
  12. by applying different weight between bias unit and input;

  13. XOR: exclusive or; XNOR: not(XOR)

  14. Multi-Class Classification:

  15. multiclass classification using neural networks is an extension of one vs. all method.

  16. MATLAB: fmincg(better for large no. of features)

  17. Symmetry Breaking
