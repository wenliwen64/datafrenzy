Definition:

$$Precision(rare cases) = \frac{\# true\_positives}{\# predicted\_positives}$$

;

$$Recall(rare cases) = \frac{\# true\_positives}{\# actual\_positives}$$

High precision / recall means good classifier. if we cheat using predict all of the patient have no cancer, then the recall will be high. - Tradeoff precision and recall(how to control): - improve hypothesis output threshold(0.5->0.9): improve precision, decrease the recall - lower threshold,: avoid false negatives - like ROC/ plot precision vs. recall; - how to decide->
